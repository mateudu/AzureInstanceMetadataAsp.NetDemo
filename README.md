# Azure Instance Metadata + ASP.NET Core API demo
Quick demo for app integration with **Azure Instance Metadata**. This demo allows you to check whether app instance is running inside or outside Azure.

## How to
Build an app. App should return information about environment at `http://APP_URL:APP_PORT/api/environment`

Read my **blog post** [HERE](https://cloudcooking.pl/index.php/2018/01/20/trust-but-verify-jak-sprawdzic-czy-maszyna-wirtualna-w-azure-jest-naprawde-w-azure/).
